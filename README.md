# N-Chain-Answers
9 Chain is a popular puzzle game in China, this program can do it.
# Build
## In Linux or Mac
```g++ main.cpp -o main```
## In Windows
Install Mingw or Mingw-w64,and do ```g++ main.cpp -o main```
# Run
Run main or main.exe.
# How to use
Input the number of chain,
input start,
input end.
